## Feature details

### Protocol tab
PICO input (Population/Intervention/Comparator/Outcome/Study design), inclusion criteria list,
exclusion criteria list (add/remove items), extraction field editor (11 default HGSOC fields + custom).

### Search tab
Query bar with PubMed/OpenAlex source toggle, runs searches and displays card grid of results,
deduplication by DOI (exact) and normalised title (lowercase+strip punctuation), import-to-pool
with count feedback.

### Screening tab (S1 title/abstract, S2 full-text)
Card interface showing title/authors/year/source/abstract, include/exclude/uncertain buttons.
Filter pills: All | Include | Exclude | Uncertain | Pending.
Progress bar: decided / total.
"Enzo screens all" button — iterates pending papers sequentially using srScreenPaper (Llama-3.1-8B),
shows AI suggestion in muted text (user can accept or override with own button).
Abort button stops batch mid-run.

### Extraction tab
Left: list of S2-included papers with status indicators.
Right: form with all extraction fields, text areas per field, Enzo pre-fill button
(runs srExtractPaper on abstract → auto-populates empty fields), Save button.
Evidence table view (toggle): grid of papers × fields.

### Synthesis tab
Live PRISMA 2020 SVG flow diagram — identification/records found, deduplication count,
S1 screened/excluded, S2 full-text/excluded, included count — all computed live from paper decisions.
Evidence table with sticky headers, TSV export (downloads extraction data as tab-separated file).
Enzo draft Methods and Enzo draft Results buttons — stream ~300 and ~400 word sections respectively,
abort available, markdown-rendered output.

### Example data
Pre-populated HGSOC review on PARPi resistance with 4 landmark papers:
Ledermann 2014 (SOLO1), Swisher 2017 (ARIEL2), Sakai 2008 (BRCAness signatures),
González-Martín 2019 (PRIMA). Shown when store is empty per project convention.
