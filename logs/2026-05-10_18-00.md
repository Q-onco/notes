### 1. `app/src/lib/tiptap-extensions.ts`
Added two new Tiptap node extensions at end of file:

**MermaidBlockExtension** (`mermaidBlock` node):
- Atom block node, draggable
- Custom NodeView renders mermaid diagrams via dynamic `import('mermaid')`
- mermaid initialized with `theme: 'neutral'` on each render
- Double-click opens `prompt()` to edit diagram source
- InputRule: typing ` ```mermaid ` auto-inserts node with example HGSOC-TME diagram
- Delete button (×) visible on hover
- Error state renders `<pre class="mermaid-err">` with the raw code in red

**DetailsExtension** (`details` node):
- Content-bearing block node with `content: 'block+'`, `isolating: true`
- Attributes: `open` (bool), `summary` (string)
- Custom NodeView: toggle button (▼/▶) + editable summary `<span>` in header
- Content is always in ProseMirror document — `display: none` when collapsed
- Toggle button updates `open` attribute via `updateAttributes`
- Summary editable inline (contenteditable span), `blur` saves to attrs
- `stopEvent` + `ignoreMutation` guards prevent ProseMirror from interfering with summary edits
