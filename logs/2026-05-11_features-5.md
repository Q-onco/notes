## Features implemented

### 1. Note Knowledge Graph (NoteGraph.svelte)
- Force-directed SVG simulation (no D3): repulsion, spring edges, center gravity, alpha cooling
- Node size = sqrt(word count); node colour = note label colour
- Edges from `href="note:ID"` links scanned in note bodies
- Click node → open note; drag node → pin & reposition
- "Graph" button added to Notes header (NotesHome.svelte)

### 2. Version History (Editor.svelte)
- Fetches last 12 commits for `data/notes.enc` from GitHub API
- For each commit: fetches file content, base64-decodes, AES-256-GCM-decrypts, finds current note by ID
- LCS word-level diff (O(n·m), capped at 2500 chars) shown in a split panel
- Clock icon in editor toolbar; history panel below analysis panel

### 3. PDF Annotator (PdfAnnotator.svelte)
- Dynamic import of pdfjs-dist v5.6.205; canvas rendering at scale 1.4
- Prev/next page navigation; 5 annotation types (Note, Hypothesis, Result, Flag, Important)
- `onInsert(text, type, page)` → inserts callout block at cursor with `[PDF p.N]` prefix
- Toolbar button appears only when note contains `<div data-type="embed-block" type="pdf">`
- If multiple PDFs in note, dropdown picker shown first

### 4. AI Inline Continuation (RichEditor.svelte + groq.ts)
- `continueWriting(contextBefore, selectedText, onChunk, signal)` added to groq.ts
- Bubble menu "Continue →" button: streams Llama 8B text inserted at cursor
- Spinner shown while streaming; click again to abort via AbortController

### 5. LaTeX Export (export.ts)
- `exportNoteTex(note)`: full preamble (geometry, amsmath, graphicx, hyperref, verbatim, soul, xcolor)
- `domToTex(node)`: handles all Tiptap custom node types + standard HTML
- Math inline → `$...$`, math block → `\[...\]`; callout → quote; table → tabular
- Checkmark-doc icon in editor toolbar
