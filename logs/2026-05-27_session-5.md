# Session 5 — 2026-05-27

## Summary
Built the **Grant Writing Studio** — a full-cycle grant application environment added as a new `grantwriter` module.

## What was built

### New component: `GrantWriter.svelte` (~1070 lines)
- **Landing**: card grid of saved grant applications, status badges, deadline countdown in days; opens HGSOC R01 example when store is empty
- **New grant form**: 6 agency cards (NIH R01/R21, ERC Starting Grant, DFG Sachbeihilfe, Wellcome Discovery, Custom) with colour coding; title, mechanism, call ID, deadline, duration fields
- **Aims tab**: CRUD for individual specific aims (label, title, hypothesis, rationale, approach, milestone, paper references); Enzo streams NIH Specific Aims 1-pager (~600 words) via `streamSpecificAims`, accept copies to Sections
- **Sections tab**: 2-column nav/editor — agency-specific section templates scaffold on creation; live word-count progress bar (green/yellow/red vs limit); Enzo drafts any section via `streamGrantSection`, accept/discard flow; preview mode renders markdown via `marked`
- **Budget tab**: line-item table (category dropdown, description, Year 1/2/3 number inputs, justification); auto-totals per year and grand total; add/remove rows
- **Review tab**: reviewer comment CRUD (reviewer name, criterion, score 1–9, full comment); Enzo drafts per-comment resubmission response (~150 words) via `streamReviewerResponse`; resolved toggle; full mock study section critique streamed via `streamGrantCritique` (impact score, criterion scores, strengths, weaknesses, resubmission suggestions)
- **Preview tab**: assembled full application text in markdown preview, total word count, copy-all to clipboard
- **HGSOC example**: pre-filled R01 on "Macrophage–CAF Crosstalk in PARP Inhibitor Resistance in HGSOC" — 3 aims (SPP1+ macrophages scRNA-seq/spatial, iCAF-macrophage JAK-STAT3 organoid tricultures, CSF1Ri+olaparib PDO+syngeneic), filled Specific Aims text, 4 budget lines; `isExample` flag blocks all mutations

### Infrastructure changes
- `types.ts`: 7 new types — `GrantAgency`, `GrantAppStatus`, `GrantAppAim`, `GrantAppSection`, `GrantAppBudgetLine`, `GrantAppReviewComment`, `GrantApp`
- `github.ts`: `grantApps: 'data/grant-apps.enc'` added to PATHS
- `groq.ts`: 4 new streaming functions — `streamGrantSection`, `streamSpecificAims`, `streamGrantCritique`, `streamReviewerResponse`; removed duplicate `streamGrantCritique` that was accidentally left from an earlier stub
- `store.svelte.ts`: `GrantApp` import, `'grantwriter'` View, `grantApps`/`grantAppsSha` state, loadAll Promise.all extended to 26 items, logout reset, `saveGrantApps()` method
- `Shell.svelte`: import + routing
- `Sidebar.svelte`: NEW_VIEWS + nav item "Grant Studio"
- `Help.svelte`: Grant Writing Studio entry in NEW_FEATURES + ALL_MODULES; section count 15→16

## Bugs fixed during build
1. **CSS `:global()` multi-selector** — Svelte 5 requires exactly one selector per `:global()`. Fixed by splitting `.draft-prose :global(h1,h2,h3,h4)`, `.sec-preview :global(h1,h2,h3,h4)`, and `.critique-output :global(h1,h2,h3,h4)` into four separate rules each.
2. **Duplicate `streamGrantCritique`** — an old single-section critique stub at groq.ts:830 collided with the new full mock-study-section version appended at the end. Old stub removed.
