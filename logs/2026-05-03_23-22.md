## Architecture decisions
- Svelte 5 + TypeScript + Vite → GitHub Pages
- GitHub PAT = auth key = AES-256-GCM encryption key (PBKDF2 600k iterations)
- Every save = GitHub commit (SHA-based atomic writes)
- Groq LLM (llama-3.1-8b-instant quick / llama-3.3-70b-versatile deep)
- Open-Meteo for weather (no key required) — Heidelberg + Chennai
- Cloudflare Worker: LLM proxy, Whisper proxy, PubMed/bioRxiv/Nature/Cell proxy
- Auto light/dark by hour (06:00–19:00 light)
- Enzo persona: warm oncology expert named after Dr. Amritha's late dog
