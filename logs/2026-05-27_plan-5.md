# Plan: Grant Writing Studio
Date: 2026-05-27

## Goals
Full-cycle grant application environment: aims builder → section-by-section Enzo drafting →
budget table → reviewer response → preview/copy. Covers NIH R01/R21, ERC Starting Grant,
DFG Sachbeihilfe, Wellcome Discovery, and Custom agencies.

## Feature scope (v1)
1. Grant list landing with HGSOC NIH R01 example (PARPi/macrophage crosstalk)
2. New grant form: agency picker, title, mechanism, call ID, deadline, duration
3. Aims tab: aims builder (label/title/hypothesis/rationale/approach/milestone), Enzo Specific Aims draft → accept to Sections
4. Sections tab: per-agency section nav, textarea editor, word count vs limit, Enzo draft + accept flow
5. Budget tab: line-item table category × Year 1/2/3, auto-totals
6. Review tab: reviewer comment cards (criterion/score/comment), per-comment Enzo response draft, mock study section critique
7. Preview tab: assembled text, total word count, copy all
