# Plan: Systematic Review Conductor — full PRISMA workflow
Date: 2026-05-27

## Goals
Full PRISMA-guided systematic review module — from protocol to synthesis.
Compresses 3-6 months of manual SR work into hours.

## Feature scope (v1)
1. Review list / landing with HGSOC example review
2. Protocol setup: PICO + inclusion/exclusion criteria + extraction fields
3. Search: PubMed + OpenAlex queries → paper pool, deduplication by DOI+title
4. Screening Stage 1 (title/abstract): card-by-card include/exclude/uncertain
   - Enzo pre-screen all (batch, llama-3.1-8b-instant)
   - Filter by decision, progress bar
5. Screening Stage 2 (full-text): same interface for S1-included papers
6. Data extraction: per-paper form (fixed HGSOC fields + custom), Enzo pre-fill
7. Synthesis: PRISMA 2020 flow SVG, evidence table (TSV export), Enzo draft methods+results
