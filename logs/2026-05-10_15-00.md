# Session log — 2026-05-10 15:00

## Feature: Massive notes editor upgrade

### Changes made

**worker/qonco-worker.js**
- Added `POST /embed-meta` endpoint: smart URL metadata extraction supporting YouTube (extracts videoId), Twitter/X (extracts tweetId), PDF detection, and generic OG-tag scraping (title, description, image, favicon, hostname)

**app/package.json**
- Added `@tiptap/extension-image`, `@tiptap/extension-subscript`, `@tiptap/extension-superscript`, `katex`

**app/src/lib/tiptap-extensions.ts** (new file)
- `createImageBlock(opts)` — Extends Tiptap Image with a NodeView: alignment controls (left/center/right/full), width slider, caption field, hover toolbar. Includes ProseMirrorPlugin for paste-to-upload and drag-drop-to-upload of images → R2
- `createAudioClipExtension()` — Inline audio player node. Stores r2Key + durationSec + label. NodeView renders a styled chip with HTML5 Audio playback
- `createEmbedBlock()` — URL embed node. NodeView renders YouTube iframe, PDF iframe, or generic link-preview card
- `createAttachmentBlock()` — File attachment chip node with download link
- `CalloutExtension` — Content-bearing callout block (info/warning/important/hypothesis/result) with coloured left border
- `InlineMathExtension` — Inline KaTeX rendering, double-click to edit formula. InputRule for `$...$` pattern
- `BlockMathExtension` — Display KaTeX block, double-click to edit. InputRule for `$$...$$`
- `ColumnExtension` + `ColumnsExtension` — Two-column CSS flex layout

**app/src/components/RichEditor.svelte** (complete rewrite)
- Two-row toolbar: row 1 (history, text format, headings, lists, blocks, alignment, code+math, decoration), row 2 (structural inserts, media inserts, utilities)
- Added: Subscript, Superscript, TextAlign, Color, Highlight multicolor, Callout picker dropdown, InlineMath, BlockMath, Columns, Image file-picker, File attach picker, URL Embed (calls /embed-meta), Find & Replace panel (Ctrl+H, replace/replace-all with match count), Text colour picker (10 swatches + clear)
- All existing extensions preserved
- `slashRef` extended with: `insertAudioClip`, `insertEmbed`, `insertAttachment`, `insertCallout`, `insertMath`, `insertColumns`
- `onUpload` prop for R2 upload delegation

**app/src/components/Editor.svelte**
- `uploadFile()` helper using `store.workerBase`
- `$effect` to set `window.__qonco_worker` for NodeView access
- Inline audio recording: start/stop with MediaRecorder, uploads blob to R2 under `note-clips/`, inserts AudioClip node. Animated recording indicator button
- Citation dialog: PubMed search (calls `/pubmed`), results listed, click inserts `[Author et al., Year](pubmed-url)` inline hyperlink
- Both wired to new toolbar buttons in editor-toolbar
